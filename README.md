# Landing Page Pessoal
Oi! Eu sou o Dom Henrique!

Desde que iniciei os meus estudos em programação, notei que todo bom desenvolvedor possuía uma página pessoal. Pensando nisso, decidi aplicar todos os meus conhecimentos em front-end e criar a minha própria landing page.

# Processo de criação
Confesso que foi a minha primeira experiência com landing pages - até então só havia desenvolvido pequenas páginas de séries e filmes que gostava. Infelizmente não criei um projeto no Figma para o esboço do site, algo que atrasou bastante o desenvolvimento.

Utilizando HTML5 e CSS3, o protótipo foi inspirado em uma estética Dark Premium Old Money, com ênfase em tons escuros.

![Antiga versão](assets/images/Docs/Old.png)

Recentemente, aplicando os meus novos aprendizados de marketing digital e web design, o projeto sofreu uma reformulação brusca. Agora as cores transmitem uma personalidade próxima do usuário, juntamente com uma tipografia clean e não-serifada. A ideia central não é mostrar minhas **hard skills**, mas também **soft skills** e a minha **personalidade**.

![Nova versão](assets/images/Docs/New.png)

***Curiosidade:*** Se você for na seção Minhas Obras e tocar em uma das imagens, irá tocar a música **Me Encontra**, da banda *Charlie Brown Jr.*.

# Planos futuros
Está sendo planejado desenvolver responsividade à página. Até o presente momento, o único tamanho é de 440px-840px. Além disso, implementar o JavaScript para mais animações está sendo fortemente cogitado.
