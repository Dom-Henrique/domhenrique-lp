Sobre mim

Acredito que a Matemática e a Física são as linguagens fundamentais do universo, e é nessa busca pela precisão que encontro minha identidade como desenvolvedor. Utilizo o C para afiar minha lógica e o Java para estruturar sistemas robustos, tratando cada linha de código como uma tradução da ordem natural das coisas. Para mim, o desenvolvimento backend não é apenas funcionalidade, mas a construção de uma base sólida, eficiente e matematicamente elegante.

Aliando essa força técnica ao meu olhar como Designer de Autoridade, utilizo HTML, CSS e Python para dar vida e presença a essas estruturas. Meu objetivo é transformar a complexidade em interfaces que comunicam confiança e impacto imediato. Como desenvolvedor Fullstack, busco a harmonia entre o rigor da engenharia e a estratégia do design, entregando soluções que não apenas resolvem problemas, mas refletem a autoridade que cada projeto merece.
